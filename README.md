## About This Project
The **RWeb Solutions Operations Monitoring Dashboard** is an internal tool designed to give the team a clear, real-time
view of every client project — from initial proposal all the way through to final delivery.
Built on Laravel with a warm rose/cream UI, it replaces messy spreadsheets with a structured, interactive pipeline
tracker.
---
## What It Tracks
- 📋 **Proposal Phase** — Stage progress (Sitemap → Homepage → All Pages → Final Homepage)
- 🎨 **UI/UX** — Designer assignments, status, and due dates
- ⚙️ **Development Phase** — Front-end and back-end assignments, progress percentages, and statuses
- 📅 **Due Dates** — Color-coded overdue notifications for UI/UX, Dev, and Final delivery dates
- ⊞ **Final Remarks** — Closing notes and delivery status per client
---
## Features
- ✅ Inline editing — click any cell to edit directly
- 🔍 Search and filter by status, stage, or keyword
- 🗑️ Recycle Bin — soft-delete with restore support
- 📋 Activity Log — tracks every change made
- 📤 Export to XLSX and PDF
- 🌙 Dark / Light theme toggle
- ⚠️ Overdue popup alert on load
