## About This Project

This is a web-based **Operations Monitoring Dashboard** built with Laravel — designed to track client projects across the full web development pipeline.

It gives teams a clear, real-time view of where every project stands, from initial proposal to final delivery.

## What It Tracks

- Proposal status per client (Sitemap, Homepage, All Pages)
- UI/UX and Developer assignments
- Front-end and back-end progress
- Final delivery dates and schedule status
