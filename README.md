<details open>
<summary><b>🌸 About this project</b></summary>

<br>

The **RWeb Solutions Operations Monitoring Dashboard** is an internal tool designed to give the team a clear, real-time view of every client project — from initial proposal all the way through to final delivery.

Built on Laravel with a warm rose/cream UI, it replaces messy spreadsheets with a structured, interactive pipeline tracker that keeps every stakeholder aligned.

</details>

---

<details>
<summary><b>📊 What it tracks</b></summary>

<br>

- 📋 Proposal Phase (Sitemap → Homepage → All Pages → Final Homepage)  
- 🎨 UI/UX assignments, status, and deadlines  
- ⚙️ Development progress (frontend & backend)  
- 📅 Due dates with overdue indicators  
- 📝 Final remarks and delivery status  

</details>

---

<details>
<summary><b>✨ Features</b></summary>

<br>

- ✅ Inline editing (spreadsheet-style)  
- 🔍 Search and filtering  
- 🗑️ Recycle bin (restore deleted data)  
- 📋 Activity logs  
- 📤 Export to Excel & PDF  
- 🌙 Dark / Light mode  
- ⚠️ Overdue alerts  

</details>
