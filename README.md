<details open>
<summary><b style="color:#c9637a;">About this project</b></summary>

<br>

The **RWeb Solutions Operations Monitoring Dashboard** is an internal tool designed to give the team a clear, real-time view of every client project — from initial proposal to final delivery.

Built with Laravel and inspired by a warm **rose/cream interface**, it replaces spreadsheets with a structured, interactive monitoring system.

</details>

---

<details>
<summary><b style="color:#c9637a;">What it tracks</b></summary>

<br>

• Proposal stages (Sitemap → Homepage → All Pages → Final Homepage)  
• UI/UX assignments, progress, and timelines  
• Development tracking (frontend & backend)  
• Due dates with overdue indicators  
• Final remarks and delivery status  

</details>

---

<details>
<summary><b style="color:#c9637a;">Features</b></summary>

<br>

• Inline editing (spreadsheet-style)  
• Smart search and filters  
• Activity logs for tracking changes  
• Recycle bin with restore support  
• Export to Excel and PDF  
• Dark / Light mode toggle  
• Overdue alerts on load  

</details>
