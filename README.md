## ✨ Overview

The **RWeb Solutions Operations Monitoring Dashboard** is a powerful internal system that transforms scattered spreadsheets into a **structured, real-time workflow tracker**.

> 💡 Track every client project — from proposal to deployment — in one clean interface.

## ⚡ Key Features

<details>
<summary>📋 Click to expand features</summary>

### 🧩 Project Tracking
- Proposal stages (Sitemap → Homepage → All Pages → Final Homepage)
- Centralized client monitoring

### 🎨 UI/UX Management
- Designer assignments
- Status tracking
- Due dates with alerts

### ⚙️ Development Monitoring
- Frontend & Backend tracking
- Progress percentages
- Status updates

### 📅 Smart Deadlines
- Color-coded overdue indicators
- Auto alerts on load

### 🛠️ Productivity Tools
- Inline editing (spreadsheet-style)
- Advanced search & filters
- Activity logs

### ♻️ Data Safety
- Soft delete (Recycle Bin)
- Restore functionality

### 📤 Export Options
- Excel (XLSX)
- PDF reports

### 🌙 UI Experience
- Dark / Light mode toggle
- Clean rose/cream aesthetic

</details>

## 🧠 Why This Exists

Managing multiple client projects manually is messy.

This dashboard solves:
- ❌ Lost progress in spreadsheets  
- ❌ No real-time visibility  
- ❌ Poor accountability  

✅ Replaced with a **live, structured pipeline system**
